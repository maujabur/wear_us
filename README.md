Wear-us
=======

Wear-us é um projeto de criação colaborativa para prototipar uma peça de computação vestível, vestível inteligente, vestimenta computacional ou qualquer outro nome que se dar em português para o tema de wearables. Para isso, acreditamos ser necessário um trabalho em equipe com mentes diversificadas, engajadas, experimentais, analíticas e mãos incansáveis, atenciosas, inquietas e criativas.
